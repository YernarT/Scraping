# Scraping Assignment

> 2022 年 4 月份 实习任务, **该爬虫案例仅供学习使用**

## 项目介绍

- `backend`: FastAPI 后端服务器, 用于爬取数据
- `frontend`: React 前端网页, 用于展示数据

## 结果报告 (Result)

演示地址: [yona-scraping.netlify.app](yona-scraping.netlify.app)

### 基本概况

爬取网站: [https://astana.spravker.ru](https://astana.spravker.ru)

- 总数据数: **1000+**
- 抓取数据类型:
  - 办公室
  - 商店
  - 餐厅
- 抓取字段:
  - `name`
  - `address`
  - `site`
  - `phone`
